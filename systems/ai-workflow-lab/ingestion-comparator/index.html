AI Workflow Lab Tool

Document Ingestion Comparator

Upload a PDF, DOCX, or HTML document and compare Markdown output from multiple ingestion adapters.

This production-facing tool uploads your document to the AI Workflow Lab backend, runs selected parsers, and returns short-lived Markdown result links.

Upload

Choose a document and parsers

Parser selection

Select up to three parsers. Some adapters may be accepted by the API but return skipped until enabled in the backend.

Privacy and retention

Uploaded documents and generated Markdown are document-derived sensitive artifacts. Upload links expire after 15 minutes; result metadata expires logically after 24 hours.

Guidance

How to read the output

What to inspect

  • Heading hierarchy and section boundaries.
  • Table preservation and list formatting.
  • Missing, duplicated, or repeated page text.
  • Parser warnings and unsupported-format failures.

What not to assume

  • AI comparison is advisory, not ground truth.
  • Parser outputs can contain extraction errors.
  • Result links are temporary and should not be shared broadly.
  • Expired results must be regenerated with a new upload.
