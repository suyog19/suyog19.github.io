Practical Machine Learning Foundations

Build, evaluate, and critique a predictive model through honest baselines, disciplined evaluation, and error analysis.

No cohort date or fee is currently promised. Register interest to help validate demand and receive a future update.

Register interest

Who this course is for

Learners who can independently clean, explore, and visualise tabular datasets and want to move from descriptive analysis into predictive modelling.

Expected prerequisites

Confident tabular data work, reproducible Python workflows, descriptive statistics, and the ability to explain analytical limitations.

Provisional learning scope

  • Prediction questions, targets, features, and units of analysis
  • Data splitting, leakage prevention, and meaningful baselines
  • Preparation pipelines and a bounded set of supervised models
  • Metric selection, validation, comparison, and error analysis
  • Overfitting, uncertainty, limitations, and responsible critique
  • An end-to-end, reviewable predictive-modelling project
