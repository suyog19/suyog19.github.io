Is this the right course for me?

Choose this course if you can write and debug small Python programs independently and want a faster, application-oriented pace. Complete programming beginners should start with Python Foundations.

Prerequisites

You should be comfortable with variables, control flow, functions, collections and files, and able to diagnose common errors in a small program.

Outcomes

The primary outcome is applying Python to data, machine-learning workflows and engineering exercises. Detailed objectives will be published before applications open.
